== Description ==
The animated responsive portfolio plugins gives your WordPress personal blog or a corporate website power to have an elegant responsive portfolios where you can showoff your amazing work or images.

You can add portfolio items from the <b>Dashboard > Portfolio > Add New</b> UI.
 
And you can display this portfolio in your website's front-end using the shortcode:
`[pre-portfolio column="3" width="400px" height="400px"]`

* You can check a live demo of this portfolio on <a href="http://wordpressdeveloper.co.in/test/portfolio/" target="_blank">Portfolio Plugin Demo</a>.

== Installation ==
1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
